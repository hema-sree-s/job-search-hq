Simple formatting, applied when you Preview or export PDF: a line starting with # becomes your name/title, ## becomes a section header (EXPERIENCE, EDUCATION, SKILLS...), and - becomes a bullet point. Everything else is a normal paragraph. Example:
# Jane Doe — Software Engineer
## Experience
- Built X, improving Y by Z%

"Export as PDF" opens a formatted preview in a new tab and triggers your browser's print dialog automatically — choose Save as PDF as the destination. This uses your browser's own built-in PDF export rather than a third-party tool, which is far more reliable.

Import from Resume

Upload your resume as a Word document (.docx) — not a PDF. Word documents extract far more reliably (real paragraphs and words, not just positioned pixels), so this pulls clean text into your Documents tab and auto-fills email/phone if found. It's a starting point, not a finished profile — review and edit after. Your original file is also saved here (up to ~700KB) so you can re-download it later. If your resume is currently a PDF, open it in Word (or Google Docs → File → Download → Word) and export it as .docx first. Want an actual PDF? Build it in Resume Builder or Documents \u2192 Clean ATS Resume, then use "Export as PDF" there.
